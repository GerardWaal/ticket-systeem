# Ticket Systeem

Basisproject voor een ticketsysteem met een gescheiden backend en frontend.

## Structuur

- `backend/` voor API en businesslogica
- `frontend/` voor de gebruikersinterface
- `docs/` voor functionele en technische documentatie

## Eerste vervolgstappen

1. Kies de stack voor backend en frontend.
2. Werk de kernfunctionaliteiten uit: tickets aanmaken, toewijzen, statusbeheer en opmerkingen.
3. Voeg authenticatie, validatie en logging toe.
4. Richt CI/CD en testautomatisering in.
